## 🔹 프로젝트 개요

- **프로젝트명**: PositivePositive  
- **장르**: 2D 로프 액션  
- **플랫폼**: PC  
- **개발 기간**: 11일  
- **개발 형태**: Unity 2D 팀 프로젝트  

---

## 🔹 주요 기능 (보스 담당)

### SlugBoss 
- 슬러그 보스의 상태 제어: 공격, 사라짐 등  
- 일정 주기마다 타겟 방향 공격 시도  
- 충돌 판정 및 HP 처리 포함  
- `Animator` 및 `Rigidbody2D` 활용한 모션 연동

### Zoom UI 연출
- 보스 등장 시 화면 확대 연출  
- UI 전환 및 카메라 연출 타이밍 제어
